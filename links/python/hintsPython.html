ModuleNotFoundError: No module named 'kivy'

Problem: The system was not finding the Kivy library even it being installed following the instructions for Installation on Windows

Detected Error: When executing the basic "Hello World!" script available on the kivy.org, I was facing "no module found" message as below:


            Traceback (most recent call last):
            File "c:\myScripts\GitHub\python\kivy\main.py", line 1, in 
                import kivy
            ModuleNotFoundError: No module named 'kivy'

Solution: The solution was to install all the following packages at this sequence, one by one:


            pip install --upgrade pip wheel setuptools --user
            pip install docutils pygments pypiwin32 kivy.deps.sdl2 kivy.deps.glew
            pip install kivy.deps.gstreamer
            pip install kivy.deps.angle
            pip install kivy
