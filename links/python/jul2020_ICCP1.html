Jogo do Nim (Nim Game)

On this project was requested to create a game from the bottom. This game is the "Nim game" that consists on having a set of pieces on the table and two players, alternately, remove a number of pieces from the table.

There is a way to win the game that is always to leave, to the oponent, a number of pieces multiple of (m+1), where m is the maximum number of pieces allowed to be removed in a single turn per player.

For this project, the rules stated that the computer will always win independently of you choose a single match or a championship (three matches).
