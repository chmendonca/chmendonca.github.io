Book: "Curso Intensivo de Python"

This book contains a general course information of Python programming. With this book is possible to learn about types of variable, lists, dictionaries, flow controls (while, for, if/else, etc), functions, exceptions, an introduction of unit tests and projects to exercise.

Project 1 - Pygame:

On this project the book had instructions to build a game "Alien Invasion". The game used a Python library Pygame that already have to many built in functions to support the games construction.

The "Alien Invasion" consists in a fleet of aliens that tryes reach the soil and a spaceship that tryes to protect against this invasion. The player controls the spaceship moving it horizontally (left-right) and shots the aliens. Every time that the aliens fleet is destroyed, a new one is placed on and it moves a little faster than previous one. The player has three attempts to avoid the invasion. The game finishes when the aliens touch the soil or they hit the spaceship.

In order to improve my skills, in parallel to this game construction, I've built another version of the same game that I called "Hero Combat". To do this other version I used a picture of a Hero that my son painted during COVID-19 pandemy. On this game, the Hero shots serynges with vacine in a random COVID-19 cloud that advances over him.
