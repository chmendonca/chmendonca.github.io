Orientação a Objetos com Java (OO with Java)

On this course I've learned the object orientation concepts on Java to start building games using "Eclipse". Some of the course contents (but not limited to) were attributes, methods (constructors, getters and setters), coupling, CRC cards modelling, basic UML, functions override, introduction to tests and TDD using "JUnit4", heritage, interfaces, access modifiers and basic on errors handling.

The game on video below is the final project of the course. Shuffled words are showed up and are compared with the player answers.
